---
title: OpenCode
---

Use Pluxx in OpenCode for wrapper generation plus attachable refinement runs.

Recommended prompt:

```text
Use Pluxx to generate this plugin, run refinement passes through OpenCode, and verify final wrapper output.
```

Install target:

```bash
npx @orchid-labs/pluxx install --target opencode
```

Permissions note:

- OpenCode permissions are compiled to native tool-level `config.permission` entries.
- Selector-specific precision (for example exact path globs or specific MCP tools) is downgraded to coarse tool permissions and should be documented in plugin guidance.
