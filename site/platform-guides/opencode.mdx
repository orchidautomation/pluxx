---
title: OpenCode
---

Use this target when you want wrapper-based packaging and optional attached runner execution.

## Build and Install

```bash
npx @orchid-labs/pluxx build --target opencode
npx @orchid-labs/pluxx install --target opencode
```

Local install path:

- `~/.config/opencode/plugins/<plugin-name>`

## Runtime Files

Key generated files under `dist/opencode/`:

- `package.json`
- `index.ts`

OpenCode packaging compiles commands, hooks, MCP wiring, and instructions into the wrapper output.

## Permissions Caveat

OpenCode permissions are compiled to native `config.permission` entries. Fine-grained selector semantics may degrade to coarse tool-level permissions; document these reductions explicitly in plugin guidance.

## Runner Attach Mode

For agent refinement using OpenCode runner:

```bash
npx @orchid-labs/pluxx agent run review --runner opencode --attach http://localhost:4096 --no-verify
```

`--attach` is only supported with `--runner opencode`.
