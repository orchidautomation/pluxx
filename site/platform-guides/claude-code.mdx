Use Pluxx in Claude Code for interactive scaffold iteration and fast refine/review loops.

Recommended prompt:

```text
Use Pluxx to scaffold this MCP, then refine taxonomy and instructions before building and installing the Claude target.
```

Install target:

```bash
npx @orchid-labs/pluxx install --target claude-code
```

If Claude Code is already open, run `/reload-plugins` in the active session after install so the new local plugin is picked up immediately.

### Slash Surface Contract

When a Claude plugin has both `commands/<name>.md` and `skills/<name>/SKILL.md`, Pluxx keeps both surfaces in the Claude bundle.

To keep the slash UX command-first without losing the semantic skill surface, Pluxx keeps the Claude command name unchanged and rewrites the colliding Claude skill with `user-invocable: false` plus a hidden internal skill name.

That keeps the command visible at its intended name while preserving the skill for Claude's automatic semantic routing.
