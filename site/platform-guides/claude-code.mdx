Use Pluxx in Claude Code for interactive scaffold iteration and fast refine/review loops.

Recommended prompt:

```text
Use Pluxx to scaffold this MCP, then refine taxonomy and instructions before building and installing the Claude target.
```

Install target:

```bash
npx @orchid-labs/pluxx install --target claude-code
```

### Slash Surface Contract

When a Claude plugin has both `commands/<name>.md` and `skills/<name>/SKILL.md`, Pluxx keeps both surfaces in the Claude bundle.

To avoid duplicate slash labels, colliding Claude command files are remapped to `commands/<name>-command.md` (for example `read-and-triage-mail-command.md`) while the canonical skill remains `skills/<name>/SKILL.md`.
