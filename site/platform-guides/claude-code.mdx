Use Pluxx in Claude Code for interactive scaffold iteration and fast refine/review loops.

Recommended prompt:

```text
Use Pluxx to scaffold this MCP, then refine taxonomy and instructions before building and installing the Claude target.
```

Install target:

```bash
npx @orchid-labs/pluxx install --target claude-code
```

### Slash Surface Contract

When a Claude plugin has both `commands/<name>.md` and `skills/<name>/SKILL.md`, Pluxx now keeps the command and omits the colliding skill from the Claude bundle.

This keeps Claude's slash UX command-first for generated workflows (for example `read-and-triage-mail`) while preserving full skills + commands output for other targets.
