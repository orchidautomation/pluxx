---
title: Codex
---

Use Pluxx in Codex for deterministic CLI-first workflows and strict review passes.

Important: in Codex, use `@pluxx` plus skills; do not rely on plugin slash commands.

Recommended prompt:

```text
Use Pluxx to import this MCP, validate the first pass, then review what is still weak before installation.
```

Install target:

```bash
npx @orchid-labs/pluxx install --target codex
```

Permissions note:

- Codex permission enforcement is external to the plugin bundle.
- Pluxx emits `.codex/permissions.generated.json` as a mirror of canonical `permissions` rules so you can wire them into Codex user/admin policy or external hooks.
