---
title: Rewrite Instructions
---

Rewrite the generated instructions section without touching user-owned custom sections.

## Flow

```bash
npx @orchid-labs/pluxx agent prepare
npx @orchid-labs/pluxx agent run instructions --runner codex
npx @orchid-labs/pluxx lint
npx @orchid-labs/pluxx test
```

## Editing boundary

Generated markdown is mixed ownership:

```md
<!-- pluxx:generated:start -->
...
<!-- pluxx:generated:end -->

<!-- pluxx:custom:start -->
...
<!-- pluxx:custom:end -->
```

Only generated blocks should be rewritten by automation.
