title: Review Scaffold
---

Run a findings-first review before shipping.

## Flow

```bash
npx @orchid-labs/pluxx agent run review --runner codex --no-verify
npx @orchid-labs/pluxx doctor
npx @orchid-labs/pluxx lint
npx @orchid-labs/pluxx test
```

## Review checklist

- taxonomy clarity
- instruction quality
- auth/setup/runtime boundaries
- host-specific correctness (especially Codex)
- examples and install flow quality
