---
title: CLI Reference
---

Core commands:

- `pluxx init --from-mcp ...`
- `pluxx doctor`
- `pluxx doctor --consumer <installed-or-built-bundle>`
- `pluxx lint`
- `pluxx eval`
- `pluxx build`
- `pluxx test`
- `pluxx mcp proxy --from-mcp ... [--record ...]`
- `pluxx mcp proxy --replay <tape.json>`
- `pluxx migrate <path>`
- `pluxx install --target ...`
- `pluxx sync [--from-mcp ...]`
- `pluxx autopilot --from-mcp ... --runner ...`

Agent commands:

- `pluxx agent prepare`
- `pluxx agent prompt taxonomy|instructions|review`
- `pluxx agent run taxonomy|instructions|review --runner claude|cursor|codex|opencode`

See [Practical handbook](/getting-started/practical-handbook) for examples.
