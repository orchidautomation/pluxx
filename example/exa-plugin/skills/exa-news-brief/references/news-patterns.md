# News Research Patterns

When the task is time-bounded:

1. calculate the exact date window first
2. search for primary announcements
3. search for independent follow-up reporting
4. search for reaction only if it adds information

Good query shapes:

- `category:news product launch April 2026 developer tools`
- `category:news company funding announcement March 2026`
- `category:news browser engine release notes reaction enterprise`

Do not treat a recent blog post as a complete timeline by itself.
