# Exa Company Research

## What To Load

- For company-specific query shapes, see [references/company-patterns.md](references/company-patterns.md).

## Workflow

1. Start with `company-scout`.
2. If the result set is large or comparative, use `source-auditor` before final ranking.
3. Prefer product pages, docs, launch posts, funding coverage, and practitioner writeups over generic directories.
4. Return the market map, strongest links, and the key differentiation notes.
