# Exa Code Research

## What To Load

- For code-specific query shapes, see [references/code-patterns.md](references/code-patterns.md).

## Workflow

1. Start with `code-scout`.
2. Prefer official docs first, then maintainer examples, then strong practitioner writeups.
3. Use `web_fetch_exa` on the best candidate URLs when snippets are not enough.
4. If several sources conflict, use `source-auditor` to explain which ones deserve trust.
