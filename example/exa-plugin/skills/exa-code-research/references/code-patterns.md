# Code Research Patterns

Use Exa to find:

- official documentation
- migration guides
- error explanations
- maintainer or practitioner examples

Good query shapes:

- `official docs for OAuth 2.1 device flow implementation`
- `maintainer migration guide react router v7 data routers`
- `production example vector database ingestion pipeline node typescript`
- `error explanation postgres relation does not exist migration`

Avoid generic tutorial spam when better primary sources exist.
