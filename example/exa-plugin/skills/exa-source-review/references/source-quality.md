# Source Quality Rules

Use these rules when the user wants ranked, curated, or "best of" results.

## Weight Higher

- practitioners doing the work
- primary product, docs, repo, or announcement pages
- independent reporting that adds evidence rather than paraphrasing
- sources with concrete examples and falsifiable claims

## Weight Lower

- generic SEO pages
- listicles without proof
- repeated commentary that cites the same thin source
- directory pages with no evidence

## Review Questions

- is this source primary, secondary, or just recycled commentary?
- does it contain evidence or only summary language?
- is it independent from the other sources we found?
- does it deserve to change the ranking?
