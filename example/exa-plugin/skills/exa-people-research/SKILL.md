# Exa People Research

## What To Load

- For people-specific search patterns, see [references/people-patterns.md](references/people-patterns.md).

## Workflow

1. Start with `people-scout`.
2. If quality thresholds matter, send the candidate set through `source-auditor`.
3. Prefer public writing, talks, repos, products, and operator signal over shallow bios.
4. Return concise structured results with URLs.
