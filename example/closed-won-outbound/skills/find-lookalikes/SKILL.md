# Find Lookalikes

Use this skill after the seed cohort is already selected.

## Workflow

1. State the similarity inputs first:
   - industry
   - company size
   - product motion
   - hiring signal
   - tech stack
   - geography
2. Use `lead.*` for candidate generation.
3. Use `research.*` to pressure-test weak or ambiguous matches.
4. Return a ranked candidate set with one-line rationales.
