# Segment Best Customers

Use this skill when the operator needs the right seed cohort before any lookalike work starts.

## Workflow

1. Start with closed-won accounts or opportunities from `crm.*`.
2. Make the selection logic explicit:
   - highest expansion potential
   - strongest retention
   - fastest time to value
   - clearest persona fit
3. Prefer a smaller defensible seed cohort over a broad noisy one.
4. Record why each account is in or out.
