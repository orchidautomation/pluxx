# Map Buying Committee

Use this skill when the target accounts are known and the next question is who to contact.

## Workflow

1. State the primary persona clearly.
2. Use `lead.*` to identify candidate contacts.
3. Use `research.*` for role validation when titles are fuzzy.
4. Exclude low-fit or stale contacts instead of padding the list.
5. Preserve role, rationale, and confidence in the output.
