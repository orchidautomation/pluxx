# Review Workflow Gaps

Use this skill when the operator or maintainer needs the honest boundary map.

## What To Cover

- native Pluxx surfaces
- required provider adapters
- placeholders or mocked v1 logic
- the smallest next step that would turn this into live proof

## Output

Return three short sections:

1. native today
2. adapter required
3. still missing for live proof
